# E-Commerce Management System

A simple E-Commerce website prototype built with Python and Flask. Displays a list of products with name, description, and price.

## ✨ Features
- **Product Catalog:** Displays 5 products including Electronics & Fashion items
- **Dynamic Rendering:** Uses Flask Jinja2 templating to render products from Python list
- **Lightweight:** Built with Flask - no database setup needed for prototype

## 🛠️ Tech Stack
- **Backend:** Python 3.x, Flask
- **Frontend:** HTML, Jinja2 Templating
- **Server:** Flask Development Server
